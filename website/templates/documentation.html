PeakPass Technical Documentation

By: Parker (Backend Developer)

What Technologies/Languages are Employed?

PeakPass employs the use of HTML, JS, and CSS for front end development. For back-end development we use Python's Flask library. Our websites are hosted with NGINX on Ubuntu Server with the Gunicorn module. For data storage we employ both SQLite for breached passwords, and a PostgreSQL database for user data.

How is Data Stored?

All login passwords are hashed using the bcrypt algorithm. All user associated information is securely encrypted using Fernet - part of the Python Cryptography library - and stored within our PostgreSQL database.

What Information is Collected?

Information collected and stored includes your login username and password, as well as any information you enter into the form for creating new items, like: name, username, password, and URL. We do not collect ANY other data, this cna be confirmed by auditing out code, available by clicking here.
