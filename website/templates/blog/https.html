HTTPS (Hypertext Transfer Protocol Secure)

By: Parker Meinershagen

What is HTTPS?

HTTPS is the Hypertext transfer protocol secure, and is the secure and encrypted version of HTTP - the main protocol of data transmission on websites. HTTPS allows websites to securely communicate with your web browser, meaning you can safely login to your accounts without leaking your data.

Any website that requires you to login using your username/password combination should use HTTPS, especially ones that are crucial to your life like: banks, insurance providers, email services, etc.

How Does it Work?

HTTPS encrypts the traffic between your web browser and the server of the website you visit using the Transport Layer Secure (TLS) protocol. The protocol uses both public and private keys for encryption, this is called asymmetric public key infrastructure. The public is available to everyone, and is not a secret, this key is used to encrypt communications. The private key is the secret and is controlled solely by the website. The private key is used to decrypt the traffic from the web browser, which allows the server to process your information like your login.

How to Know if it's Enabled

You can determine if HTTPS is in use by looking in the top left corner of your web browser, just to the left of the link. If you see a closed lock, you are securely connected to that website. However, many browsers will use a lock that is opened, or text that says "Not Secure" to alert you that you are connected using the HTTPS protocol. If you see any of these signs DO NOT enter any of your information into that website, it is not safe to do so.

Tools for Enabling HTTPS

Sometimes, websites do not automatically use the HTTPS protocol even when it is available, in order to fix this issue you should install a browser extension to automatically attempt to connect to the HTTPS versions of the websites you visit. In order to do this, visit your browser extensions download page and search for the extension called "HTTPS Everywhere". This extension automatically attempts to connect you to the secure version of all sites you visit, so that you never accidentally connect to an insecure version.
